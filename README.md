# Friendcoin--Decentralized-Blockchain-Transaction-System

📌 Friendcoin — Decentralized Blockchain Transaction System

Friendcoin is a lightweight decentralized blockchain-based transaction system designed to simulate how cryptocurrencies work. This project demonstrates core blockchain concepts such as distributed ledgers, mining, proof-of-work, and secure peer-to-peer transactions.

🚀 Features

🔗 Blockchain Structure – Linked blocks secured using cryptographic hashes

⛏️ Proof-of-Work Mining – Ensures network security and block validation

💸 Transaction System – Send and receive Friendcoin between users

🌐 Decentralized Network – Multiple nodes maintaining the blockchain

🔒 Data Integrity – Tamper-proof transaction records

📡 REST API Support – Interact with the blockchain via endpoints




🧠 How It Works

Users create transactions (sender → receiver → amount)

Transactions are grouped into a block

Miners solve a cryptographic puzzle (Proof-of-Work)

Block is added to the blockchain

Network nodes validate and sync the updated chain


🛠️ Tech Stack

Backend: Node.js / Python (Flask) (choose based on your project)

Blockchain Logic: Custom implementation

API Testing: Postman / Curl

Data Format: JSON
